# **App Name**: Expressive Reads

## Core Features:

- User Authentication: Secure user authentication using Firebase Authentication with Google and email/password options.
- Product Catalog: Browse and search for books and notebooks using the Google Books API with filters for genre, price, author, and rating.
- Material 3 Expressive Theming: Implements all requested features from Material 3 including color theming, expressive shapes, dynamic animations and delightful feedback upon user interactions.
- Shopping Cart: Manage and update items in a shopping cart with quantity adjustments and subtotal calculations, persisted using local browser storage.
- Wishlist: Add books and notebooks to a wishlist and manage them for future purchases.
- Checkout Flow: Guide users through a multi-step checkout process with address selection, payment method (mock), and order confirmation.
- Personalized Recommendations: AI-powered tool to provide personalized book recommendations based on browsing history and viewed items.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) to capture a sense of sophistication and modernity.
- Background color: Light grey (#F2F0F7), offering a muted backdrop that highlights the product imagery and textual content, while still feeling within the purple color story.
- Accent color: Magenta (#7D5260), used to add a burst of playful contrast and energy, guiding users towards key interactive elements and calls to action.
- Headline font: 'Poppins' (sans-serif) for headlines, bringing a geometric and fashionable aesthetic to headers. Body Font: 'PT Sans' (sans-serif), offering improved readability in larger blocks of text
- Custom icons with rounded shapes, aligning with the Material 3 Expressive design, to represent categories and actions.
- Generous spacing and card-based layouts to create a visually appealing and easy-to-navigate experience.
- Natural, springy animations on interactive elements like buttons, cards, and form submissions to provide delightful feedback.
